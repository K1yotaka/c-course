## 实验课 04

1. 使用递归函数计算 $1 + 2 + 3 + \dots + n$
2. 编写 `bool is_prime(int n)` 函数，实现判断整数 $n$ 是否为素数。
3. 调用第 $2$ 题中的 `is_prime` 函数，求出 $1000$ 以内的素数。
4. 编写 `void swap(int *pa, int *pb)` 函数，实现交换两个变量的值。
5. 编写 `void sort(int *pa, int *pb, int *pc)` 函数，实现将三个整数进行从大到小排序。
